Part II VLA 预训练为何更难
难点不是多模态本身，而是理解与控制的错配
核心论点

VLA 的难点，本质上不是“看懂世界”，而是要把理解性表征稳定地转化为控制性表征。

对应证据
Evidence
VLM4VLA

VLM4VLA 用少于 1% 新参数适配 24 个 VLM，并在 3 个 benchmark 上发现：VLM 初始化有益，但通用能力并不能稳定预测控制性能。

论据：作者进一步做模态消融后指出，主要瓶颈在视觉编码器而非语言模块；向视觉侧注入 control-relevant supervision 能稳定提升下游控制。

数据矛盾
互联网图文很多，但机器人闭环交互数据稀缺且昂贵。
人类视频很多，但真正可执行、可泛化的动作轨迹很少。
VLA 需要的是高价值轨迹，不是普通语义样本。
目标矛盾
VLM 的优化目标是“看懂”和“说对”。
VLA 的目标是“做对”，且必须闭环成功。
语义能力强，不等于控制能力强。
接口矛盾
语言是离散 token，动作是连续控制量。
世界状态又具有动态、时序与物理约束。
因此难点集中在 中间接口设计。

关键前提：VLA 的问题不是“VLM 能不能迁移到机器人”，而是 VLM 学到的表征能否被转化为控制所需的表征。

理解与控制的错配，是后续演进的起点

Part III 预训练对象在演进
主线不是模型做大，而是表征对象迁移
Evolution

VLA 预训练的主线：从语义表征，走向动作表征，再走向世界动态表征

Stage 1
语义预训练

主要结论：VLM 预训练为 VLA 提供稳定起点，但它带来的是理解先验，不是可直接执行的控制先验。

论据：VLM4VLA 显示：VLM 初始化整体优于从零训练，但标准 VLM 能力和控制表现相关性弱；问题集中在视觉表征与控制需求之间的 domain gap。

Stage 2
动作预训练

主要结论：动作预训练的关键不是直接重建视频变化，而是从人类视频中提取可执行的动作先验。

论据：CLAP 用对比学习把视频 latent 对齐到机器人 proprioceptive latent 与量化 codebook，显式过滤 visual entanglement；其 CLAP-RF 还能以 183ms 推理实现高频精细操作。

Stage 3
世界动态预训练

主要结论：更高层次的预训练对象不再只是动作本身，而是动作如何驱动未来状态演化。

论据：VideoVLA 联合预测动作与未来视频，并观察到 imagined future 质量与动作可靠性强相关；VLA-JEPA 则用 leakage-free latent prediction 学到对 camera motion 与背景扰动更鲁棒的 dynamics abstraction。

Semantic
VLM4VLA

证明“VLM 强”不等于“VLA 强”

Action
CLAP

把人类视频压到可对齐的动作先验空间

Dynamics
VideoVLA

生成式动态先验比纯理解先验更利于泛化

Latent WM
VLA-JEPA

在潜空间预测未来，避免像素噪声与信息泄漏

核心结论：VLA 预训练的主线，不是单纯把 backbone 做大，而是预训练对象从语义表征逐步迁移到动作表征，再迁移到世界动态表征。

预训练对象的迁移，决定了领域演进方向

Part IV 为什么世界模型开始变重要
动作学习进一步走向 dynamics learning
Two Representative Routes

世界模型的价值，不一定在生成逼真视频，而在学到更控制友好的动态表征

Video Generator Route
VideoVLA

主要结论：视频生成器中学到的物理动态先验，可以直接转化为机器人操控的泛化能力。

论据：论文明确指出，高质量 imagined futures 与 reliable action prediction、task success 强相关；其泛化能力来自预训练视频生成模型 + 动作/视觉双预测的共同约束。

Latent World Model Route
VLA-JEPA

主要结论：对控制更有价值的不是未来像素，而是 future latent states 中蕴含的 action-relevant transition semantics。

论据：VLA-JEPA 明确分析了 pixel tethering、nuisance motion 与 information leakage 三类失真来源，并用 leakage-free JEPA 预训练在 LIBERO、SimplerEnv 与真实操作中获得一致鲁棒增益。

从动作表征到世界-动作耦合表征
理解当前场景
预测动作会怎样改变状态
学习 future state 的演化规律
形成可控的 dynamics prior

这一步意味着：VLA 预训练开始不再只关心“当前帧里看到了什么”，而开始关心动作与未来世界变化之间的稳定关联。

核心判断
VideoVLA 说明：joint video-action forecasting 可以把“想象力”变成动作可靠性的内在指标。
VLA-JEPA 说明：latent-state prediction 比 pixel reconstruction 更接近控制所需的动态语义。
因此 world model 的价值开始从“可视化未来”转向“提供可控的 dynamics prior”。
世界模型的价值，从生成能力转向 dynamics representation

Part V 真正的瓶颈在 action/world interface
不是模型不够大，而是接口吃不下更多信息
Bottleneck Thesis

当前最核心的扩展瓶颈，越来越不像是 backbone 本身，而像是动作表征与生成接口

视觉编码器越来越强
动作信息必须经过 tokenizer / codebook
动作生成还要满足实时延迟与内存约束
最终可扩展性被 interface 锁死
一句话结论

真正的瓶颈不是“模型不够大”，而是 action interface 既决定模型能学到什么，也决定模型最终能跑到什么程度。

Information Bottleneck
The Compression Gap

主要结论：离散动作 tokenization 会把 encoder scaling 的收益截断在 codebook 处。

论据：同样升级视觉编码器时，Diffusion Policy 在 LIBERO 上提升超过 21 个百分点，而 OAT 增益明显受抑；放大 codebook 容量后，encoder sensitivity 才部分恢复。

Training Friendliness
ActionCodec

主要结论：好 tokenizer 的标准不在重建误差，而在是否让 VLA 更容易学、更不易过拟合。

论据：ActionCodec 提出 temporal overlap、词表去冗余、增强多模态互信息、token 独立性四个原则；在 LIBERO 上，SmolVLM2-2.2B 可达到95.5%，强化版达97.4%，且无需 robotics pre-training。

Systems Constraint
Characterizing VLA Models

主要结论：action generation 是边缘 VLA 落地的主瓶颈，而且瓶颈属性是 memory-bound，不是 compute-bound。

论据：论文测得动作生成占端到端延迟最高 75%；当前平台距离 10Hz 实时控制仍有200-300 倍差距，5 倍算力提升只带来约 1.4 倍延迟改善。

表征层问题
离散化压缩损失信息
训练层问题
token 稳定性决定优化难度
系统层问题
动作生成延迟卡住实时部署
瓶颈正在从模型规模转向接口设计

Part VI 预训练不是终点，而是闭环系统起点
pretrain → post-train → online adapt

未来真正有效的 VLA 体系，不是单点式预训练，而是 表征学习 + 世界模型 + 后训练 + 数据引擎 + 在线适应 的闭环协同。

表征学习

先把语义、动作、世界动态的中间表征学起来

世界模型

提供未来状态、短时模拟与 latent dynamics

后训练

对策略进行 RL 或 latent alignment 强化

数据引擎

自动生产、筛选、回流高价值轨迹

在线适应

部署后持续修正，不再假设模型静态不变

RL Post-Training
WoVR

结论：不必假设世界模型完美，只要显式控制 hallucination，WM 就能成为可靠 simulator。
论据：LIBERO 从 39.95% 提升到 69.2%，真实机器人从 61.7% 提升到 91.7%。

Latent Post-Training
World2Act

结论：WM→VLA 的 post-training 不应依赖 pixel-space supervision，而应对齐 latent dynamics。
论据：其 LLM skill decomposition 支撑任意长度 rollout，真实世界性能提升 6.7%。

Subgoal Guidance
AtomVLA

结论：长任务失败的核心在 instruction grounding gap，需要子任务级检查点。
论据：LLM 分解 atomic subtasks，latent world model 对 action chunk 打分；LIBERO 达 97.0%，LIBERO-PRO 达 48.0%。

Continual Learning
Simple Recipe Works

结论：大模型时代的 VLA 可能天然具备 continual learning 属性。
论据：论文发现简单的 Seq. FT + LoRA + on-policy RL 就能同时保持 plasticity、低遗忘与 zero-shot generalization。

Sim + Data Engine
Simulation Distillation / Seed2Scale

结论：未来 scaling 不只是模型训练，还包括 simulator 与 data engine 的协同。
论据：SimDist 只微调 dynamics，15-30 分钟真实数据即可快速适应；Seed2Scale 仅用 4 条示教启动，目标模型相对提升 209.15%（22.18%→68.57%）。

这一组论文共同说明
世界模型的价值越来越像训练基础设施，其核心作用是提供 latent rollout、reward、subgoal 或 dynamics prior。
能力形成机制正在从一次性 pretrain 转向 post-train + simulator + online update 的组合。
数据引擎开始从“被动数据集”转向“主动生产、主动验证、主动回流”的闭环部件。
对未来路线的直接判断
不是简单的 end-to-end real-world scaling。
更可能是 simulation pretraining + world model adaptation + real-world correction。
再叠加自进化数据引擎，形成持续提升的系统级增长机制。
下一阶段能力提升将更多来自系统闭环

Part VII 四个核心判断
Evidence-backed Conclusions
01

VLM 的语义能力，不能直接等价为 VLA 的控制能力

代表证据：VLM4VLA。24 个 VLM、3 个 benchmark 的结果共同表明：VLM 能力分数不是控制表现的可靠预测器，视觉编码器才是主要瓶颈。

02

VLA 预训练主线，是从语义预训练走向动作预训练，再走向世界动态预训练

代表证据：CLAP、VideoVLA、VLA-JEPA。从 executable latent actions 到 joint video-action forecasting，再到 leakage-free latent state prediction，预训练对象在持续升级。

03

当前最核心的扩展瓶颈，不只是模型规模，而是 action/world interface

代表证据：The Compression Gap、ActionCodec、Characterizing VLA Models。离散 codebook、token 稳定性与 memory-bound generation 共同构成了学习与部署的双重上限。

04

下一阶段的能力提升，主要来自系统级闭环，而不只是一次性预训练

代表证据：WoVR、World2Act、AtomVLA、Simple Recipe Works、Simulation Distillation、Seed2Scale。world model、post-training、continual adaptation 与 data engine 正在形成闭环系统。

结尾版核心表述

未来更强的 VLA，不会只是一个更大的模型，而会是一个更完整的系统。
