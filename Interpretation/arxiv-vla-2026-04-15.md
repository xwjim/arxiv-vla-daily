## Goal2Skill: Long-Horizon Manipulation with Adaptive Planning and Reflection

- **arXiv**: <https://export.arxiv.org/abs/2604.13942v1>
- **ID**: `2604.13942v1`
- **分类**: cs.RO
- **作者**: Zhen Liu, Xinyu Ning, Zhe Hu, Xinxin Xie, Weize Li, Zhipeng Tang, Chongyu Wang, Zejun Yang, Hanlin Wang, Yitong Liu, Zhongzhu Pu
- **总结状态**: 已总结
- **标签**: VLA, 长时程操作, 分层规划, 反思恢复, 结构化记忆, 机器人操作
- **PDF**: [Goal2Skill Long-Horizon Manipulation with Adaptive Planning and Reflection.pdf](../pdfs/Goal2Skill Long-Horizon Manipulation with Adaptive Planning and Reflection.pdf)

### 一、论文元数据

- **标题：** Goal2Skill: Long-Horizon Manipulation with Adaptive Planning and Reflection
- **核心机构：** 北京邮电大学；InspireOmni AI；清华大学
- **一句话本质：** 单体端到端 VLA 很难稳住长时程操作，这篇论文的核心思路是把 **高层记忆-规划-反思** 与 **低层几何执行** 解耦，形成一个可闭环纠错的双系统框架。

### 二、破局点 (The Gap)

- **已有共识/做法：** 现有 VLA 在短时窗、局部感知充分的 manipulation 任务上已经较强，通常直接基于最近观测做端到端动作预测。
- **核心痛点：** 一旦任务变成长时程、部分可观测、带遮挡且存在多阶段依赖，单体策略会缺少持久记忆、显式分解和失败恢复能力，容易出现“局部动作合理、全局任务失败”。
- **本文切入点：** 作者把问题拆成两个系统，高层负责语义规划与反思纠错，低层负责精细执行，从而让记忆与控制各自发挥优势。

### 三、核心机制 (How it works)

- **输入 -> 输出：** 全局任务目标 + 当前观测 -> 高层生成结构化子任务与约束 -> 低层 VLA 执行连续动作 -> 执行结果反哺高层做验证与重规划。
- **关键模块 1：** **高层 planner**：由 VLM 驱动，维护结构化任务记忆，完成 goal decomposition、结果验证和 error-driven correction。
- **关键模块 2：** **低层 executor**：由 VLA 驱动，通过 diffusion-based action generation 执行子任务，并利用 geometry-preserving filtered observations 聚焦关键结构。
- **关键模块 3：** **闭环反思机制**：每一步执行后，高层根据后置条件和记忆判断是否达成目标，若失败则调整当前子任务或重写后续计划。

### 四、核心认知与发现 (Key Insights)

- **长时程操作首先是“记忆与纠错”问题：** 真正让 VLA 掉队的往往不是单步控制精度，而是能否记住中间状态、发现失败并及时修正。
- **规划与执行解耦很关键：** 高层语义推理和低层几何控制若塞进同一个单体模型，往往两边都做不稳；明确分层反而更鲁棒。
- **结构化记忆确实带来增益：** 在 RMBench 上从最强 baseline 的 9.8% 提升到 32.4%，说明“记忆 + 闭环恢复”不是装饰模块，而是核心能力来源。

### 五、致命弱点与演进方向 (Critical Analysis)

**🛑 核心局限 (Limitations)：**

- **整体成功率仍不高：** 32.4% 虽然明显优于 baseline，但离可部署的长时程操作能力还有很大差距。
- **高层验证质量决定上限：** 如果 planner 的后置条件判断或失败诊断不可靠，闭环就可能把错误放大。

**⚠️ 论证瑕疵：**

- 摘要强调对 RMBench 的显著提升，但没有展开说明不同任务长度、不同失败类型下，收益是否均匀存在。

**🚀 下一步高价值方向 (Next Steps)：**

- **把反思机制做得更可证：** 提升 post-condition verification 的可靠性，减少高层误判导致的错误重规划。
- **扩展到更开放场景：** 检验这种双系统设计在真实机器人、多任务切换和更长任务链上的可扩展性。

## Jump-Start Reinforcement Learning with Vision-Language-Action Regularization

- **arXiv**: <https://export.arxiv.org/abs/2604.13733v1>
- **ID**: `2604.13733v1`
- **分类**: cs.LG, cs.AI, cs.RO
- **作者**: Angelo Moroncelli, Roberto Zanetti, Marco Maccarini, Loris Roveda
- **总结状态**: 已总结
- **标签**: VLA, 强化学习, PPO, 稀疏指导, 样本效率, sim2real
- **PDF**: [Jump-Start Reinforcement Learning with Vision-Language-Action Regularization.pdf](../pdfs/Jump-Start Reinforcement Learning with Vision-Language-Action Regularization.pdf)

### 一、论文元数据

- **标题：** Jump-Start Reinforcement Learning with Vision-Language-Action Regularization
- **核心机构：** 瑞士南方应用科学与艺术大学（SUPSI/IDSIA）；瑞士意大利语区大学（USI）；米兰理工大学
- **一句话本质：** 这篇论文不是让 VLA 直接接管控制，而是把它当成 **训练早期的稀疏教师信号**，用来给 RL 提供方向性偏置，从而更快学会高频精细操作。

### 二、破局点 (The Gap)

- **已有共识/做法：** RL 擅长高频闭环控制，但在稀疏奖励、长时程任务中常常探索低效、credit assignment 很差；VLA 有高层语义先验，但直接做高速精细控制又不够稳。
- **核心痛点：** 两类方法各有长板却难以互补，要么 RL 从零学太慢，要么直接蒸馏/模仿 VLA 又容易把教师缺陷一并继承。
- **本文切入点：** 让 VLA 只在训练早期、低频率地给出“往哪个方向做动作更合理”的建议，而不是持续接管策略。

### 三、核心机制 (How it works)

- **输入 -> 输出：** 语言目标 + 视觉观测 -> VLA 给出稀疏动作建议 -> PPO 在环境交互中结合 regularization 更新策略 -> 最终得到可超越教师的 RL 策略。
- **关键模块 1：** **VLAJS 框架**：将 VLA 视作 transient high-level guidance source，用于 bias early exploration。
- **关键模块 2：** **directional action-consistency regularization**：不是强行 imitation，而是软性约束 RL 动作方向与 VLA 建议一致。
- **关键模块 3：** **稀疏且退火的教师调用**：VLA 指导只在训练早期出现，并逐步 anneal，让 agent 后续依赖自身在线适应能力。

### 四、核心认知与发现 (Key Insights)

- **VLA 最有价值的阶段可能是训练早期：** 它未必适合作为最终控制器，但非常适合帮 RL 走出探索冷启动。
- **软对齐优于硬模仿：** 方向一致性 regularization 保留了 RL 的自由探索空间，因此更有机会最终超过教师策略。
- **多模态先验能显著改善样本效率：** 若干任务中交互量减少超过 50%，并且在真实 Franka Panda 上展现了 zero-shot sim-to-real transfer。

### 五、致命弱点与演进方向 (Critical Analysis)

**🛑 核心局限 (Limitations)：**

- **依赖教师质量：** 如果 VLA 的建议方向本身偏差较大，早期 regularization 可能把 RL 引向错误探索区域。
- **验证范围仍偏 manipulation：** 摘要覆盖的是六个典型操作任务，尚未说明在更复杂长时程任务或真实长期在线学习中是否同样成立。

**⚠️ 论证瑕疵：**

- 摘要突出样本效率提升，但未说明不同任务对指导频率、退火策略和教师模型质量的敏感度。

**🚀 下一步高价值方向 (Next Steps)：**

- **自适应控制指导强度：** 根据训练阶段与不确定性动态决定何时向 VLA 请求建议，而不是固定退火。
- **扩展到更难真实场景：** 检验该框架在长时程、接触丰富、奖励更模糊的真实机器人任务中的稳定性。

## Vision-and-Language Navigation for UAVs: Progress, Challenges, and a Research Roadmap

- **arXiv**: <https://export.arxiv.org/abs/2604.13654v1>
- **ID**: `2604.13654v1`
- **分类**: cs.RO
- **作者**: Hanxuan Chen, Jie Zheng, Siqi Yang, Tianle Zeng, Siwei Feng, Songsheng Cheng, Ruilong Ren, Hanzhong Guo, Shuai Yuan, Xiangyue Wang, Kangli Wang, Ji Pei
- **总结状态**: 已总结
- **标签**: UAV, 视觉语言导航, 综述, VLA, 世界模型, 具身智能
- **PDF**: [Vision-and-Language Navigation for UAVs Progress, Challenges, and a Research Roadmap.pdf](../pdfs/Vision-and-Language Navigation for UAVs Progress, Challenges, and a Research Roadmap.pdf)

### 一、论文元数据

- **标题：** Vision-and-Language Navigation for UAVs: Progress, Challenges, and a Research Roadmap
- **核心机构：** 道通智能（Autel Robotics）；南京大学；威斯康星大学麦迪逊分校；南方科技大学；香港大学；北京大学
- **一句话本质：** 这篇论文系统梳理了 UAV-VLN 从传统模块化方法走向 **agentic system、VLA 与 world model 融合** 的演进路径，并给出面向真实部署的研究路线图。

### 二、破局点 (The Gap)

- **已有共识/做法：** 地面机器人 VLN 已有较成熟的发展脉络，但 UAV-VLN 涉及连续三维运动、复杂户外感知和算力受限平台，问题结构明显不同。
- **核心痛点：** 该方向缺少一份把任务定义、方法谱系、仿真器/数据集/评测体系和真实部署瓶颈统一串起来的系统综述。
- **本文切入点：** 作者从 formal task definition 到方法学演进，再到资源生态与开放挑战，构建了一套面向 UAV-VLN 的完整认知框架。

### 三、核心机制 (How it works)

- **输入 -> 输出：** 领域文献与 benchmark 生态 -> 方法学 taxonomy + 资源综述 + 挑战分析 -> 面向未来的 research roadmap。
- **关键模块 1：** **方法谱系梳理**：从早期 modular/deep learning 方法，一路梳理到 foundation model 驱动的 agentic system、VLM、VLA 与 generative world model 融合路线。
- **关键模块 2：** **资源生态综述**：系统回顾 simulators、datasets 和 evaluation metrics，帮助研究者理解这个领域的公共基座。
- **关键模块 3：** **部署瓶颈分析**：集中讨论 sim-to-real gap、动态户外环境感知、语言歧义推理，以及大模型在资源受限 UAV 平台上的高效部署问题。

### 四、核心认知与发现 (Key Insights)

- **UAV-VLN 正在经历范式迁移：** 研究重心正在从分离式感知-规划流水线，转向 foundation model 驱动的统一 embodied intelligence 框架。
- **空中导航的独特难点不能照搬地面 VLN：** 连续 3D 动作空间、户外动态扰动和飞行平台算力/能耗限制，使 UAV-VLN 成为更硬核的系统问题。
- **未来高价值方向已经相当明确：** 多机协同、空地协作，以及 world model 与 VLA 的深度结合，是这个领域最值得持续跟进的前沿。

### 五、致命弱点与演进方向 (Critical Analysis)

**🛑 核心局限 (Limitations)：**

- **综述不直接提供新方法验证：** 论文的主要价值在于结构化整理与路线指引，而非通过新 benchmark 或新模型解决问题。
- **路线图天然会快速过时：** foundation model 驱动的 embodied AI 发展极快，survey 的前沿判断需要持续更新。

**⚠️ 论证瑕疵：**

- 摘要提出了多个核心挑战和未来方向，但未说明这些方向在短中期内的优先级与相互依赖关系。

**🚀 下一步高价值方向 (Next Steps)：**

- **把路线图转成可执行 benchmark 设计：** 为多机协同、空地协同和真实户外语言导航构建更具区分度的评测体系。
- **推进轻量化与真实部署闭环：** 让大模型导航能力与 UAV 端侧约束真正对齐，而不是停留在仿真或离线分析层面。
